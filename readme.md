### Link365 Global Solutions- Technical Exam

Exam is created using Laravel and VueJS.
Things that are provided or used in the repository.
  - Migration
  - Seeder
  - Env
  - Cache

#Setup  
```
clone repository
composer install
npm install
set .env specially ur db
```

#Start
```
php artisan serve
npm run watch
```

#Migration
```
php artisan migrate
php artisan db:seed
```
